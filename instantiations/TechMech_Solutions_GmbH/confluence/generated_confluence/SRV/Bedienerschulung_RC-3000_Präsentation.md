# Bedienerschulung RC-3000 Präsentation

## Einführung
Die Bedienerschulung für die Roboterzelle RC-3000 ist entscheidend für die sichere und effiziente Nutzung der Automatisierungstechnik. Diese Präsentation bietet einen Überblick über die Funktionen, Sicherheitsvorkehrungen und Wartungsanforderungen der RC-3000.

## Zielgruppe
Diese Schulung richtet sich an:
- Bediener der Roboterzelle RC-3000
- Technisches Personal
- Wartungsteams

## Technische Spezifikationen der RC-3000

| Spezifikation             | Wert                     |
|---------------------------|--------------------------|
| Max. Traglast             | 500 kg                   |
| Reichweite                 | 1200 mm                  |
| Wiederholgenauigkeit      | ±0,05 mm                 |
| Programmierungsschnittstelle | Teach-in und grafische Benutzeroberfläche |
| Antriebssystem            | Servomotoren             |
| Spannungsversorgung        | 400 V, 3-Phasen          |

## Sicherheitsvorkehrungen
- **Not-Aus-Schalter**: Jeder Arbeitsbereich der RC-3000 ist mit einem Not-Aus-Schalter ausgestattet, der im Notfall sofort betätigt werden muss.
- **Sicherheitsabstände**: Halten Sie einen Sicherheitsabstand von mindestens 2 Metern zur Roboterzelle ein, wenn diese in Betrieb ist.
- **Schutzkleidung**: Tragen Sie stets geeignete Schutzkleidung, einschließlich Gehörschutz und Sicherheitsschuhe.

## Bedienung der RC-3000
1. **Start der Roboterzelle**
   - Überprüfen Sie alle Sicherheitsvorkehrungen.
   - Schalten Sie die Hauptstromversorgung ein.
   - Aktivieren Sie die Roboterzelle über das Bedienfeld.

2. **Programmierung**
   - Wählen Sie den gewünschten Modus (Automatik/Manuell).
   - Nutzen Sie die grafische Benutzeroberfläche zur Programmierung der Bewegungsabläufe.
   - Führen Sie einen Testlauf durch, um die Programmierung zu überprüfen.

3. **Wartung**
   - Regelmäßige Inspektionen alle 500 Betriebsstunden.
   - Schmierung der Gelenke gemäß den Herstellerangaben.
   - Austausch von Verschleißteilen alle 1000 Betriebsstunden.

## Beispielprogrammierung
- **Bewegungsablauf**: 
    - Startpunkt: (X: 0 mm, Y: 0 mm, Z: 0 mm)
    - Endpunkt: (X: 500 mm, Y: 0 mm, Z: 200 mm)
    - Geschwindigkeit: 300 mm/s
    - Beschleunigung: 400 mm/s²
