# DMS-100 Kalibrieranleitung

## Einleitung

Die Kalibrierung des DMS-100 Qualitätsprüfsystems ist entscheidend für die Gewährleistung der Messgenauigkeit und der Produktqualität. Diese Anleitung beschreibt die Schritte zur Kalibrierung des Systems gemäß den technischen Spezifikationen und Best Practices.

## Benötigtes Equipment

- DMS-100 Qualitätsprüfsystem
- Kalibrierungsreferenzstandard (z. B. Kalibriergewichte)
- Digitales Multimeter (DMM)
- Laptop mit installierter DMS-100 Software
- Kalibrierungsprotokoll (siehe Anhang)

## Kalibrierungsverfahren

### 1. Vorbereitung

1. **Systemüberprüfung**
   - Stellen Sie sicher, dass das DMS-100 System ordnungsgemäß installiert und funktionsfähig ist.
   - Überprüfen Sie die Verbindungen und die Stromversorgung.

2. **Umgebungsbedingungen**
   - Kalibrieren Sie das System bei Raumtemperatur (20 °C ± 2 °C) und einer relativen Luftfeuchtigkeit von 40% bis 60%.

### 2. Kalibrierungsschritte

#### 2.1. Mechanische Kalibrierung

| Schritt | Beschreibung                                              |
|---------|----------------------------------------------------------|
| 1       | Überprüfen Sie die mechanische Ausrichtung der Sensoren. |
| 2       | Justieren Sie die Sensoren mithilfe von Kalibriergewichten. |
| 3       | Dokumentieren Sie die Justierungen im Kalibrierungsprotokoll. |

#### 2.2. Elektronische Kalibrierung

1. **Software starten**
   - Starten Sie die DMS-100 Software auf dem Laptop.

2. **Daten erfassen**
   - Führen Sie eine Nullkalibrierung durch:
     - Messen Sie einen Referenzwert von 0 g.
     - Bestätigen Sie den Wert in der Software.

3. **Referenzkalibrierung**
   - Wählen Sie ein Kalibriergewicht (z. B. 500 g) und führen Sie die Messung durch.
   - Vergleichen Sie den gemessenen Wert mit dem Referenzwert.
   - Dokumentieren Sie den gemessenen Wert und die Abweichung.

### 3. Abgleich und Nachjustierung

- Sollte eine Abweichung von mehr als 0,5% festgestellt werden, führen Sie eine Nachjustierung des Systems durch.
- Führen Sie die Kalibrierung erneut durch, um die Genauigkeit zu überprüfen.

## Abschluss der Kalibrierung

1. **Protokollierung**
   - Füllen Sie das Kalibrierungsprotokoll vollständig aus und speichern Sie alle relevanten Daten.

2. **Systemtest**
   - Führen Sie einen abschließenden Systemtest durch, um die korrekte Funktionalität zu gewährleisten.

3. **Freigabe**
   - Nach erfolgreicher Kalibrierung kann das DMS-100 System zur Qualitätsprüfung eingesetzt werden.

### Hinweis

Die Kalibrierung sollte regelmäßig, mindestens jedoch einmal jährlich, durchgeführt werden, um die langfristige Genauigkeit des Systems sicherzustellen. Bei Änderungen der Umgebungsbedingungen oder nach Reparaturen ist eine sofortige Kalibrierung erforderlich.
