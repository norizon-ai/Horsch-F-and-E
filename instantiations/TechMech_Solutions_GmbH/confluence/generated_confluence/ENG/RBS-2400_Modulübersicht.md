# RBS-2400 Modulübersicht

## 1. Einleitung
Das RBS-2400 Modul ist eine innovative Lösung von TechMech Solutions GmbH, die speziell für die Automatisierung in der Fördertechnik entwickelt wurde. Es kombiniert fortschrittliche Technologie mit modularen Komponenten, um eine flexible und effiziente Förderlösung zu bieten.

## 2. Technische Spezifikationen

| Spezifikation          | Wert                      |
|------------------------|---------------------------|
| Fördergeschwindigkeit   | bis zu 2 m/s              |
| Tragfähigkeit           | max. 500 kg               |
| Antrieb                 | Elektrischer Antrieb      |
| Steuerung               | SPS-gestützt              |
| Energieverbrauch        | 1,5 kW                    |
| Abmessungen (L x B x H)| 2400 mm x 800 mm x 950 mm |

## 3. Moduleigenschaften

### 3.1 Modulares Design
Das RBS-2400 Modul zeichnet sich durch sein modulares Design aus, welches eine einfache Anpassung und Erweiterung ermöglicht. Es kann problemlos mit anderen Modulen aus der RBS-Serie kombiniert werden, um spezifische Anforderungen zu erfüllen.

### 3.2 Fördertechnologie
Die Fördertechnologie des RBS-2400 basiert auf einem robusten Riemenantrieb, der eine gleichmäßige und leise Förderung gewährleistet. Die Riemen sind aus hochfestem Material gefertigt und bieten eine lange Lebensdauer sowie geringe Wartungsanforderungen.

### 3.3 Sicherheitsmerkmale
Das Modul ist mit mehreren Sicherheitsfunktionen ausgestattet, darunter:
- Not-Aus-Schalter an mehreren Stellen
- Überlastschutz
- Sicherheitslichtschranken zur Erkennung von Personen und Objekten

## 4. Anwendungsbereiche
Das RBS-2400 Modul eignet sich für diverse Branchen, darunter:
- **Automotive**: Transport von Bauteilen in der Montage
- **Lebensmittel**: Hygienische Förderung von Verpackungen
- **Pharma**: Automatisierung in der Verteilung von Arzneimitteln

## 5. Beispielkonfiguration
Ein Beispiel für eine typische Konfiguration könnte wie folgt aussehen:
- **Anzahl der Module**: 3
- **Gesamtlänge**: 7200 mm
- **Tragfähigkeit pro Modul**: 400 kg
- **Steuerungssystem**: Siemens S7-1200
